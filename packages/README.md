# WOFI v2 packages

This folder is the npm workspaces root for WOFI v2 modules.

Workspaces are configured in the repo-root `package.json`.

Packages:
- `packages/kernel/` — `@wofi/kernel` (Module A1)
- `packages/store/` — `@wofi/store` (Module A3)
- `packages/indexer/` — `@wofi/indexer` (Module A4)
- `packages/query/` — `@wofi/query` (Module A5)
